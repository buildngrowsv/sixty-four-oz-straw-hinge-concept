# 64 oz insulated bottle · gravity-hinged straw concept

Static concept package: interactive 2D comparison (cap flex vs rigid straw vs mid-hinge), inline SVG mechanisms diagram (large silicone sock on cut rigid stubs), reference PNG board, and design notes for short-span silicone joints.
